# Análisis de señales con Transformada de Fourier
Este repositorio contiene el desarrollo de una actividad académica sobre el análisis de señales en el dominio del tiempo y en el dominio de la frecuencia utilizando Python.
## Objetivo
Analizar señales elementales mediante la Transformada de Fourier, observando su comportamiento en el dominio del tiempo y en el dominio de la frecuencia.
## Señales analizadas
- Pulso rectangular
- Función escalón
- Señal senoidal
## Herramientas utilizadas
- Python
- Google Colab
- NumPy
- Matplotlib
- GitHub
## Desarrollo
En el código se generan señales elementales y se grafican en el dominio del tiempo. Después, se aplica la Transformada Rápida de Fourier mediante la función `np.fft.fft()` para obtener la magnitud y la fase del espectro de frecuencia.
También se analizan propiedades importantes de la Transformada de Fourier, como:
- Linealidad
- Desplazamiento en el tiempo
- Escalamiento
## Conclusión
La Transformada de Fourier permite analizar una señal desde el dominio de la frecuencia, identificando las componentes que no siempre son evidentes en el dominio del tiempo, esta herramienta es útil en áreas como telecomunicaciones, procesamiento de audio, electrónica, vibraciones y análisis digital de señales.
